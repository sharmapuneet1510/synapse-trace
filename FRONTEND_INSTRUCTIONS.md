# Frontend Instruction Set
## Premium Red-and-White Enterprise UI with Liquid Display Feel

Use the reference only for the red-and-white color direction. Do not copy its layout or structure.

In addition to the color theme, the UI should have a refined **liquid display feel** with **smooth curved edges**, soft flowing surfaces, and premium modern depth. The goal is to blend enterprise seriousness with polished contemporary softness.

## Visual Direction
The interface should feel:
- premium
- modern
- fluid
- elegant
- executive-friendly
- clean
- high-end

Think of a corporate enterprise platform with a subtle “liquid glass / liquid surface” influence, but still practical and professional.

## Color Rules
Use:
- strong corporate red as the primary accent
- white as the dominant base
- soft gray and light neutral tones for separation
- dark gray / near-black for text

Keep the palette limited and disciplined.

## Shape Language
Use smooth, curved geometry throughout the UI:
- rounded containers
- soft corner radii
- flowing section edges
- gently curved cards
- polished panels
- subtle layered surfaces

Avoid sharp boxy layouts unless needed for dense data tables.

Recommended corner style:
- cards and panels: medium to large radius
- buttons: rounded but not pill-shaped unless intentional
- modals and drawers: soft premium rounded edges
- inputs: gently rounded corners

## Liquid Display Feel
Create a subtle liquid-inspired presentation by using:
- smooth visual flow between sections
- soft gradients only where needed
- layered surfaces with gentle elevation
- translucent or glass-like treatment in selected premium areas
- soft highlights and refined shadows
- rounded section transitions
- elegant spacing that makes the layout feel breathable and fluid

Do not overdo the effect.
This should not look playful, futuristic, or overly glossy.
It should remain enterprise-grade and believable.

## Layout Guidance
The layout should still be structured and professional, but visually softened through:
- curved section containers
- rounded cards
- soft visual grouping
- smooth alignment
- balanced whitespace
- elegant spacing rhythm

Use a 4/8 spacing system.

## Component Styling
### Buttons
- Primary: solid red with white text, rounded corners, premium depth
- Secondary: white with red border and red text, same soft curve language

### Cards
- white or very light neutral surfaces
- subtle shadow or border
- curved edges
- spacious padding
- clean structure

### Tables
- keep tables readable and enterprise-focused
- use softer outer container corners
- keep internal cell structure clean and restrained
- use red only for active or important states

### Inputs
- rounded corners
- subtle borders
- clean focus rings
- polished enterprise look

### Modals / Panels / Drawers
- smooth corner radius
- layered premium surface
- elegant shadow
- spacious content layout

## Typography
Use clean sans-serif typography.
The curved/liquid look should come from the surfaces and shapes, not from decorative text styling.
Typography should remain crisp, structured, and highly readable.

## Restrictions
Do not make it:
- cartoonish
- overly glossy
- too glassy
- neon
- playful
- consumer-app-like
- futuristic sci-fi

This is still an enterprise platform first.

## Final Goal
Build an original enterprise UI that combines:
- a premium red-and-white corporate palette
- smooth curved edges
- subtle liquid-display-inspired surface styling
- clean whitespace
- readable typography
- structured professional layouts

The end result should feel like a next-generation executive enterprise platform with softness and elegance, without losing trust or clarity.
